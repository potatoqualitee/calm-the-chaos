Keyword Groups

Enable or disable entire groups or individual keywords. Changes are saved automatically.

Custom Keywords

Add your own keywords to filter. These will be saved separately from the default groups.

Matching Options

Choose how strictly keywords should match content.

URL Filtering Mode

Choose whether filtering is enabled or disabled by default for all websites.
Filtering Enabled by Default
When enabled by default: Listed URLs will be EXCLUDED from filtering
When disabled by default: Listed URLs will be INCLUDED in filtering

URL Groups

Add URLs that should be excluded from filtering. Use wildcards to match variations:
gmail.com* (matches gmail.com and all its pages)
*.google.com (matches all Google subdomains)
*/messages (matches any site's message pages)

Custom URLs

Add your own URLs to exclude from filtering. Remember to use wildcards for matching variations.

Collapse Style

Choose how filtered content should be hidden from view.

Element Groups

Enable or disable filtering for different types of web elements. These control which parts of websites are checked for keywords.

Settings Backup

Export your current settings to a file for backup, or restore from a previous backup.

Remote Configuration

Import settings from a shared URL to sync with a team configuration.
