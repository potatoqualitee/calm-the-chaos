Keyword Groups

Custom Keywords

Add your own keywords to filter. These will be saved separately from the default groups.

Domain Filtering Mode

Choose whether filtering is enabled or disabled by default for all domains.
Filtering Enabled by Default
When enabled by default: Listed domains will NOT be filtered
When disabled by default: Listed domains will BE filtered

Examples: mail.*, outlook.com*, *.google.com
